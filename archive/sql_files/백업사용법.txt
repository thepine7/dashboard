========================================
HnT 데이터베이스 백업/복원 사용법
========================================

📋 필요한 프로그램:
- MySQL Client (mysqldump, mysql 명령어)
- Windows 명령 프롬프트 (cmd)

🔧 사용 방법:

1. 백업하기:
   - simple_backup.bat 파일을 더블클릭
   - 자동으로 현재 날짜/시간으로 백업 파일 생성
   - 예: hnt_backup_20241223_143022.sql

2. 복원하기:
   - simple_restore.bat 파일을 더블클릭
   - 백업 파일명 입력 (예: hnt_backup_20241223_143022.sql)
   - 확인 후 복원 실행

⚠️ 주의사항:
- 복원 시 현재 데이터베이스의 모든 데이터가 덮어써집니다
- 백업 파일은 안전한 곳에 보관하세요
- 복원 전에 반드시 현재 데이터를 백업하세요

🔍 문제 해결:
- "mysqldump가 인식되지 않습니다" 오류 시:
  → MySQL Client가 설치되어 있는지 확인
  → PATH 환경변수에 MySQL bin 폴더가 추가되어 있는지 확인

- "접근이 거부되었습니다" 오류 시:
  → 데이터베이스 연결 정보 확인
  → 네트워크 연결 상태 확인
